Open Source AI Companion

Live AI Co-Pilot For Your Screen + Voice

AI Companion watches your screen, listens to optional voice/chat input, and responds in real time with natural commentary. Built with Python + Ollama, optimized for local use.

Try The Repo Quick Start

What It Does

  • Reads your screen at fixed intervals
  • Supports local vision models via Ollama
  • Speaks responses with TTS
  • Accepts live voice input (always-listen or push-to-talk)
  • Accepts terminal chat commands while running

Quick Start

  1. Install dependencies 
    pip install -r requirements.txt
  2. Pull model + start Ollama 
    ollama pull qwen2.5vl:3b
ollama serve
  3. Run app 
    python main.py

Live Chat Commands

  • /say <message>
  • /tone hype|coach|chill
  • /silent on|off
  • /speak <n>
  • /status
  • /help
